# Composer Timeline Explorer

An interactive static website for exploring composers through time, combining biography, works, and historical context.

## Project Structure

- `index.html`: site entry point
- `assets/`: frontend styles, scripts, and images
- `data/`: structured content and UI mappings
- `docs/`: planning and design documents

## Current Scope

- Traditional Chinese UI
- Composer timeline layout
- Work, event, and context data structure
- Static-site-friendly frontend architecture

## Development Notes

- This project currently uses plain HTML, CSS, and JavaScript
- JSON data is loaded on the client side
- Run with a local server instead of opening `index.html` directly
